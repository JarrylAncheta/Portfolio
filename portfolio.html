For this project, I went for a dark background so it would place emphasis on the contrasting warm and cool colors. From Adobe Illustrator, I was able to create the gradient.

This piece of art consists of a gradient between light blue and purple. The vibrant colors help catch viewer's eyes, which was my intention.

For this project, I went for a simplistic background so it would focus the attention on the wave gradient particle. The particle consists of different lines of light blue and red, contrasting both warm and cool colors.

My interactive project features a wave-like effect which endlessly swirls in motion. Black background is utilized to emphasize the vibrant blue color. As the mouse is moved, particles create wave effects. The project is meant to sooth the viewers and help them escape reality into a whole new dimension.

This image was edited in Photoshop using several methods of color adjustments, wave effect, and halftone patterns. The aesthetic is to give a time capsule in time.
